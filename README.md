🍇 Fruit Match: Memory Card Game
A vibrant, responsive Memory Card Game built with React. Test your memory by matching pairs of delicious fruit icons. Keep track of your moves, aim for the lowest score, and try to beat your personal best!

🌟 Features
Vibrant Fruit Icons: Uses modern, clean fruit icons for a visually appealing experience.

Dynamic Score Tracker: Monitors your current attempts/turns and tracks your best (lowest) score using localStorage.

Smooth Animations: Fluid card-flipping transitions using CSS 3D transforms.

Auto-Shuffle: Cards shuffle automatically on page load or when you hit the "Restart" button.

Fully Responsive: Optimized for mobile, tablet, and desktop screens.

🛠️ Tech Stack
Framework: React (Vite / Create React App)

Icons: React Icons (or FontAwesome / custom SVG icons)

Styling: CSS Modules / Tailwind CSS

State Management: React Hooks (useState, useEffect, useRef)

🚀 Getting Started
Follow these steps to get a local copy of the game up and running.

Prerequisites
Make sure you have Node.js (v16 or higher) installed on your machine.
